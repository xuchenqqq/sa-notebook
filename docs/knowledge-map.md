# 知识地图

下面是软考系统分析师考试的核心知识结构，按模块归纳，方便快速查阅。

## 1. 系统开发与分析

- 系统开发生命周期（SDLC）
- 可行性分析
- 需求获取与需求定义
- 业务流程建模
- 系统分析方法与工具

## 2. 系统建模与设计

- 结构化分析与面向对象分析
- 数据流图、实体关系图、用例图
- 数据字典、功能分解
- 系统设计原则与模块划分

## 3. 数据库与数据管理

- 关系模型与数据库范式
- 数据库设计、表结构、索引
- SQL 基础与查询优化
- 数据一致性与事务管理

## 4. 软件工程与项目管理

- 软件过程模型（瀑布、迭代、敏捷等）
- 项目计划、资源管理、进度控制
- 风险管理与质量保证
- 文档与配置管理

## 5. 网络与安全

- 计算机网络基础
- 网络拓扑与协议
- 信息安全基础与防护策略
- 访问控制与加密技术

## 6. 系统实施与维护

- 系统测试与验证
- 系统集成与部署
- 用户培训与支持
- 运营维护与持续改进

## 7. 复习重点提示

- 重点掌握“需求分析”、“系统设计”、“数据库设计”三大核心模块
- 结合真题理解知识点如何在考试中出题
- 学会用图表归纳知识结构，提高记忆效率
